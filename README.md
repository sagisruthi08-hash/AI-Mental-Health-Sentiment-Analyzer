# AI-Mental-Health-Sentiment-Analyzer
AI-based Mental Health Sentiment Analyzer built using Python, Streamlit, and Google Colab to detect emotions from user text and provide wellness feedback.
# Features
- Detects emotions from user text
- Calculates mental wellness score
- Provides supportive wellness feedback
- Interactive web interface using Streamlit
- AI-powered emotion analysis

# Technologies Used
- Python
- Streamlit
- Transformers
- Google Colab

# How It Works
1. User enters their feelings in text form
2. AI analyzes the emotional sentiment
3. The system detects the emotion
4. A wellness score is generated
5. Supportive feedback is displayed

# Future Improvements
- Voice input support
